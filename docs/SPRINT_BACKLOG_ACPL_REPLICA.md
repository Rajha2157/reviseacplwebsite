# Sprint Backlog - ACPL Replica

## Sprint Goal
Deliver a responsive ACPL replica website with complete public page coverage, rebranded visual identity, and launch-ready QA controls.

## Story List

1. IA-01 - Global Navigation and Footer
- Implement shared navigation across all pages.
- Implement footer with About, Courses, Policies links.
- Priority: High
- Estimate: 3 points

2. HOME-01 - Homepage Content and CTA
- Build hero, training highlights, testimonials, and careers CTA.
- Ensure visual distinction from source by applying approved palette.
- Priority: High
- Estimate: 5 points

3. CRS-01 - Courses Catalog
- Build six-course catalog with pricing and short descriptions.
- Add sample PDU table for course credibility section.
- Priority: High
- Estimate: 5 points

4. ABT-01 - About and Team Pages
- Build About page with philosophy, vision, mission blocks.
- Build Team page with leadership and certification highlights.
- Priority: Medium
- Estimate: 5 points

5. CAR-01 - Careers and Jobs Flow
- Build Careers page with resume CTA.
- Build Jobs page with role/location summaries.
- Priority: High
- Estimate: 5 points

6. POL-01 - Policy Pages
- Build Privacy, Terms, Refund, and Delivery pages.
- Keep legal placeholders clearly marked for final legal signoff.
- Priority: High
- Estimate: 8 points

7. QLT-01 - Responsiveness and Accessibility
- Validate mobile, tablet, desktop layout.
- Check color contrast and readable text hierarchy.
- Priority: High
- Estimate: 5 points

## Definition of Done
- All 10 pages accessible through internal links.
- No broken internal links.
- Rebrand palette applied globally.
- Basic policy/legal placeholders present.
- QA checklist executed with no critical blockers.
